# alu-shell

## Overview
This repository contains **basic shell scripts** for learning Bash.  
It covers fundamental topics like file navigation, variables, loops, permissions, and process management.

## Folder Structure
- `basics/` – simple scripts to practice basic shell commands  
- `init_files_variables_and_expansions/` – working with variables and expansions  
- `io_redirections_and_filters/` – input/output redirection and filters  
- `loops_conditions_and_parsing/` – using loops and conditionals  
- `permissions/` – file permissions and execution  
- `processes_and_signals/` – managing processes and signals  
- `README.md` – this file  

## Requirements
- Scripts must be run on **Ubuntu 20.04 LTS**  
- Each script should be **exactly 2 lines long** (`$ wc -l file` should return 2)  
- Scripts must **end with a newline**
