# Shell Permissions

This project contains shell scripts that demonstrate how to manage
Linux file permissions, ownership, and groups.

## Tasks Covered

0. Switch current user to betty
1. Print effective username
2. Print groups current user belongs to
3. Change file owner
4. Create empty file
5. Add execute permission to owner
6. Modify multiple permissions
7. Add execute permission to everybody
8. Set special permission (James Bond)
9. Set specific permission mode
10. Mirror file permissions
11. Add execute permission to directories
12. Create directory with specific permissions
13. Change group owner
14. Change owner and group for all files
15. Modify symbolic link ownership
16. Conditional owner change
