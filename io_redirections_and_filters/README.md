This project is about Shell I/O Redirections and Filters.

Each script performs a specific task using shell commands such as:
echo, cat, head, tail, find, wc, sort, uniq, grep, tr, rev, and cut.

All scripts:
- Are written for Ubuntu 20.04 LTS
- Contain exactly two lines
- Start with #!/bin/bash
- Are executable

Tasks include:
- Displaying file contents
- Filtering text
- Counting and sorting data
- Working with special characters
- Using pipes and redirections
